# TreeToCSV

<img src="./app_icon.png" width="128" height="128" alt="TreeToCSV Icon" />

Windows 11向けにデザインされた、フォルダ構造を可視化し、チェックボックスで不要なファイル・フォルダを除外した上でCSV出力できるデスクトップアプリケーションです。

---

## 💡 アプリケーション概要
「TreeToCSV」は、指定したフォルダの階層構造をエクスプローラー風のツリー構造で表示し、チェックボックス操作によって特定のファイルやフォルダをCSVの出力対象から除外した上で、階層インデント形式のCSVファイル（Shift_JIS）を生成します。

---

## ✨ 主な機能と特徴

### 1. Windows 11 に馴染むモダンなUIデザイン
* Fluent Design を意識した角丸、洗練された配色、微細なホバーエフェクト。
* `Segoe MDL2 Assets` グリフを用いた美しいフォルダ／ファイルアイコン表示。

### 2. 親子連動する三状態（Tri-state）チェックボックス
* **親から子への連動**: 親フォルダのチェックを外すと、その配下の子フォルダ・ファイルが一括で除外（チェックオフ）されます。
* **子から親への連動**: 子要素のチェック状態に応じて、親フォルダが「全選択（True）」「全解除（False）」「部分選択（Null/一部選択）」に自動で切り替わります。

### 3. 特殊なCSV出力仕様
* **文字コード**: `Shift_JIS (CP932)`。Excel等で文字化けせずにそのまま開くことができます。
* **階層の列シフト（インデント形式）**:
  * 階層の深さに応じて項目（フォルダ名・ファイル名）を右にシフトして出力します（1階層目は1列目、2階層目は2列目...）。
  * 表の崩れを防ぐため、常に最大20列固定（カンマ数19個）でCSV行を生成します。20階層を超える分は20列目に出力されます。
* **ダブルクォーテーションエスケープのON/OFF**:
  * **ONの場合**: すべての名前を `"` で囲んで出力し、安全なCSV形式を保ちます。
  * **OFFの場合**: 名称にカンマ（`,`）が含まれていてもそのまま出力します（仕様通り列ズレを許容します）。

### 4. 🔒 セキュリティと堅牢性の向上
* **CSVインジェクション（Formula Injection）対策**: ファイル名やフォルダ名の先頭が `=`, `+`, `-`, `@` などの数式トリガー文字で始まる場合、Excel等で開いた際に不正な数式として自動実行される脆弱性を防止するため、先頭に `'`（シングルクォーテーション）を付与して安全にテキストデータ化して出力します。
* **ファイルロック対策**: 保存先CSVファイルがExcel等の別のアプリケーションで開かれており、書き込みロックが発生している場合でも、アプリが異常終了せず、「ファイルが他のプログラムで開かれている可能性があります」と具体的な原因をユーザーにダイアログで提示します。

## 🚀 使い方

1. **対象フォルダの選択**:
   * 画面上部の「参照...」ボタンをクリックし、スキャンしたいフォルダを選択します。
2. **除外設定の選択**:
   * 中央のツリービューで、CSVに出力したくないフォルダやファイルのチェックボックスをオフにします。
3. **出力設定**:
   * 下部エリアの「ファイル名・フォルダ名をダブルクォーテーション (") で囲む」チェックボックスで出力形式を設定します。
4. **CSV出力の実行**:
   * 「CSV出力」ボタンをクリックし、保存先のファイル名（例: `FolderStructure.csv`）を指定して出力します。

## 🛠️ 開発環境・ビルド手順

### 前提条件
* .NET 8.0 SDK (またはそれ以降)

### 実行方法
プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
dotnet run
```

### ポータブル版（スタンドアロン）のビルド方法
Windows向けに、.NETランタイムがインストールされていない環境でも動作する自己完結型のポータブルEXEファイルを生成するには、以下のコマンドを実行します。
```bash
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:PublishReadyToRun=true
```
ビルド完了後、`bin/Release/net10.0-windows/win-x64/publish/` ディレクトリに自己完結型の `TreeToCSV.exe` が出力されます。
